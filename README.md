# Calyko
Calyko is a simple, dynamically typed programming language.

# Building from source
## Prerequisites
* [git](https://git-scm.com/)
* [cmake](https://cmake.org/)
* A C99 compatible C compiler (gcc, clang, MSVC, etc.)

## Build instructions
Clone the repository
```shell
git clone https://github.com/WyvernAllow/calyko.git
```

Configure with CMake
```shell
cmake calyko
```

Build
```shell
cmake --build calyko --config Release
```
